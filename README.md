# OreMiner

OreMiner is a game project built using **pure vanilla Java**.

---

### **Overview**

This project is a small simulator based on the popular sandbox game, **Minecraft!** It focuses on core mechanics like resource gathering and tracking.

---

### **Requirements**

JDK: For Java, you can download [here](https://www.oracle.com/java/).

Python: For Python, you can download [here](https://python.org/downloads).
